# Text Emphasis

Various ways to emphasize and format text in Markdown.

## Bold Text

**This text is bold using asterisks**

__This text is bold using underscores__

## Italic Text

*This text is italic using asterisks*

_This text is italic using underscores_

## Bold and Italic

***This text is bold and italic***

___This text is also bold and italic___

**_Mix and match bold and italic_**

## Strikethrough

~~This text has a strikethrough~~

This is ~~deleted~~ text with ~~multiple~~ strikethroughs.

## Inserted Text

++This text is inserted++

## Marked/Highlighted Text

==This text is marked or highlighted==

You can ==highlight== important parts of a sentence.

## Combinations

You can combine **bold**, *italic*, ~~strikethrough~~, ++inserted++, and ==highlighted== text.

Here's a sentence with **bold _italic_** text.

And another with ~~**bold strikethrough**~~ text.

## Inline Code

Use `code` for inline code formatting.

Combine with other styles: **bold `code`** or *italic `code`*.

## Subscript and Superscript

Water molecule: H~2~O

Einstein's equation: E = mc^2^

Footnote reference: text^[note]^
