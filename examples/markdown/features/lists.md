# Lists

Markdown supports ordered and unordered lists with multiple nesting levels.

## Unordered Lists

- Item 1
- Item 2
- Item 3
  - Nested item 3.1
  - Nested item 3.2
    - Double nested item
    - Another double nested
- Item 4

## Ordered Lists

1. First item
2. Second item
3. Third item
   1. Nested numbered item
   2. Another nested item
4. Fourth item

## Mixed Lists

1. Ordered item
   - Unordered nested item
   - Another unordered nested
     1. Back to ordered
     2. Still ordered
2. Second ordered item
   - Mix it up again

## Lists with Rich Content

- **Bold item** with emphasis
- *Italic item* with style
- Item with `inline code`
- Item with [a link](https://example.com)
- Item with ~~strikethrough~~

## Multi-line List Items

- This is a list item that spans multiple lines.
  You can continue writing on the next line by indenting
  with two spaces or a tab.

- This item has multiple paragraphs.

  Here's the second paragraph in the same list item.
  It requires a blank line before it.

- Back to single line items.

## Definition-style Lists

- **Term 1**
  - Definition for term 1
- **Term 2**
  - Definition for term 2
  - Additional note for term 2
