# Footnotes

Footnotes allow you to add notes and references without cluttering the main text.

## Basic Footnotes

This is a sentence with a footnote[^1].

Here's another sentence with a different footnote[^2].

[^1]: This is the content of the first footnote.
[^2]: This is the content of the second footnote.

## Named Footnotes

You can use descriptive names for footnotes[^note-name].

This makes the markdown more readable[^important-note].

[^note-name]: Descriptive names help identify footnotes in the source.
[^important-note]: This is an important note about the topic.

## Multi-line Footnotes

This has a longer footnote[^long].

[^long]: This is a longer footnote that spans multiple lines.

    You can even include multiple paragraphs by indenting them.

    And continue with more content in the same footnote.

## Inline Footnotes

This is an inline footnote^[This content appears directly in the text].

Inline footnotes are useful for short notes^[Like this one].

## Footnotes with Formatting

This footnote contains formatting[^formatted].

[^formatted]: This footnote has **bold text**, *italic text*, and `code`.

## Multiple References

You can reference the same footnote multiple times[^repeated].

See the previous footnote again[^repeated].

[^repeated]: This footnote is referenced twice in the document.

## Footnotes in Context

According to research[^research], the method is effective.

The study[^study] showed significant results.

Additional data[^data] supports the conclusion.

[^research]: Smith, J. (2023). Research Methods. Academic Press.
[^study]: Johnson, A. (2023). Study Results. Science Journal, 45(3), 123-145.
[^data]: Brown, B. (2023). Data Analysis Report. Technical Report TR-2023-01.
