# Task Lists

Task lists allow you to create checklists with checkboxes.

## Basic Task Lists

- [x] Completed task
- [ ] Incomplete task
- [x] Another completed task
- [ ] Yet another incomplete task

## Nested Task Lists

- [x] Phase 1: Planning
  - [x] Define requirements
  - [x] Create mockups
  - [x] Get approval
- [ ] Phase 2: Development
  - [x] Setup project
  - [x] Implement core features
  - [ ] Add tests
  - [ ] Code review
- [ ] Phase 3: Deployment
  - [ ] Staging deployment
  - [ ] Production deployment
  - [ ] Monitor logs

## Task Lists with Rich Content

- [x] **Bold task** with emphasis
- [ ] Task with `inline code`
- [x] Task with [a link](https://example.com)
- [ ] Task with ~~strikethrough~~ text
- [x] Task with ==highlighted== text
