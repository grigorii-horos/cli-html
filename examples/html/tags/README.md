# Standard HTML Tags Examples

This directory contains examples of standard HTML tags rendered with default theme settings.

## Overview

These examples demonstrate how HTML tags are rendered in the terminal without custom inline styles. All styling comes from the default theme configuration.

## Running the Examples

```bash
# View a specific example
html examples/html/tags/headers.html

# Or run from the project root
node bin/html.js examples/html/tags/headers.html
```
