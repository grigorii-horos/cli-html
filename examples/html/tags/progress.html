Progress and Meter Examples

Progress Element

Progress bars show completion of a task with unknown total duration.

30% complete: 30%

40% complete: 40%

70% complete: 70%

Over max clamps to 100%: 100%

Missing max defaults to 1: 50%

Empty bar: 0%

Full bar: 100%

File Upload Progress

Uploading file 1 of 3: 33%

Uploading file 2 of 3: 66%

Uploading file 3 of 3: 100%

Meter Element

Meter shows a scalar measurement within a known range.

Basic Meter (0-1 range)

Disk usage: 60% (60%)

Battery: 85% (85%)

Almost empty: 15% (15%)

Full: 100% (100%)

Meter with Custom Range

Temperature: 75°C (75°C out of 100°C)

Score: 85/100 (85 out of 100)

Progress: 450 (450 of 1000 items)

Meter with Low/High/Optimum Values

CPU Usage (low is good): 30%

CPU Usage (medium): 55%

CPU Usage (high): 95%

Storage Metrics

RAM: 6.4 GB of 16 GB used

SSD: 240 GB of 512 GB used

Network: 45 Mbps of 100 Mbps

Different Fill Levels

Empty: 0%

Quarter: 25%

Half: 50%

Three quarters: 75%

Full: 100%

Custom Min Value

Rating (3-5 scale): 4.2 stars

Temperature (-20 to 40°C): 22°C

Progress vs Meter

Use Progress for: Task completion (downloads, uploads, installations)

Example: 75% Installation progress

Use Meter for: Measurements with context (disk space, ratings, temperature)
