FAQ Section

What is CLI-HTML?

CLI-HTML is a terminal renderer for HTML and Markdown with full GitHub Flavored Markdown support.

It supports syntax highlighting, customizable themes, and extensive inline customization.

Code Documentation

Function Documentation

Function name: renderHTML(html, theme)

Parameters:

  • html - HTML string to render
  • theme - Optional custom theme object

Returns: Formatted terminal output
