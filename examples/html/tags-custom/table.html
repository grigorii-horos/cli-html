Table Examples with Custom Styles

Tables support customization at multiple levels with priority cascade:

  • td/th cell level (highest priority) - data-cli-color
  • tr row level - data-cli-tr-color
  • thead/tbody/tfoot section level - data-cli-thead-color, data-cli-tbody-color, data-cli-tfoot-color
  • Theme configuration (lowest priority)
