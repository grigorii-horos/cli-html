Unordered Lists - HTML type Attribute

type="disc" (default)

  • Disc marker (•)
  • Default marker type
  • Uses theme.ul.markers.disc configuration

type="square"

  • Square marker (▪)
  • Alternative marker type
  • Uses theme.ul.markers.square configuration

type="circle"

  • Circle marker (⚬)
  • Hollow circle marker
  • Uses theme.ul.markers.circle configuration

Nested Lists with Automatic Rotation

  • Level 1 - disc (•)
    • Level 2 - square (▪)
      • Level 3 - circle (⚬)
        • Level 4 - back to disc (•)

Combining type with data-cli-* Attributes

Override marker color while keeping type:

  • Square marker with red color
  • Type defines marker, data-cli-marker-color defines color

Override both marker and color:

  • Completely custom arrow marker
  • data-cli-marker overrides type attribute

Ordered Lists - HTML type Attribute

type="1" (default)

  1. Decimal numbers
  2. Default numbering
  3. 1, 2, 3...

type="A"

  1. Uppercase letters
  2. Alphabetical ordering
  3. A, B, C...

type="a"

  1. Lowercase letters
  2. Alphabetical ordering
  3. a, b, c...

type="I"

  1. Uppercase Roman numerals
  2. Roman numbering
  3. I, II, III...

type="i"

  1. Lowercase Roman numerals
  2. Roman numbering
  3. i, ii, iii...

Nested Ordered Lists with Rotation

  1. Level 1 - decimal (1, 2, 3...)
    1. Level 2 - Roman uppercase (I, II, III...)
      1. Level 3 - Alphabetic uppercase (A, B, C...)
